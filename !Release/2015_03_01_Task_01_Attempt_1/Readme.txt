Better - релиз для сдачи задания номер 1
К релизу нет спецификации, так как версия Альфа.
Всем участникам с замиранием сердца смотреть на стену в группе MineCraft и ждать результатов.

Описание релиза:
В момент запуска в консоли появляется число: Blocks ... N - оно зависит от константы chunks.
В момент отрисовки мира каждую секунду выводится другое число M (именно мира, а не меню).
Просьба участников выполнить программы из обеих релизов и прислать эти числа, а также какое графическое ядро у вас стоит. Это позволит избежать нудного копания в спецификации OPENGL.

Ряд параметров настраивается константами:

файл: GameProcess.h
const int chunks=8; всего chunks*chunks чанков
const int repeatblock=8; период повторения блоков при размещении

файл: GameProcess.cpp
float StepIKE=0.01; скорость перемещения
float StepKKG=0.1; скорость поворота

файл: ProjectGL.h
const float WindowFogDensity=0.003;интенсивность тумана
const float WindowDepth=10000; дальняя граница окна
const float WindowAngle=1024;устанавливает масштаб изображения (только 3D)
const float WindowNear=0.1; ближняя граница окна

файл: main.cpp
int ScreenX=1024;// ширина окна в момент создания
Чтобы сохранить масштаб при изменении размера окна необхрдимо установить  WindowAngle=ScreenX
